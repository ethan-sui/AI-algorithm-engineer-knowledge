# 算法与数据结构
## 排序算法
### 比较类排序
通过比较来决定元素间的相对次序，由于其时间复杂度不能突破O(nlogn)，因此也称为非线性时间比较类排序
1、交换排序
- 冒泡排序

&emsp;&emsp;算法步骤：

&emsp;&emsp;①、比较每一对相邻的元素，从开头到结尾，如果第一个比第二个大，就交换它们两个，这样在最后的元素是最大的数

&emsp;&emsp;②、除了最后面的已经排序好的，对剩下的元素继续进行①步骤，直到排序完成
- 快速排序
