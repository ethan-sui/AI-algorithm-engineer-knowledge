## 运算符

### 1、算数运算符
- 用于处理四则运算
- \+， -， *， /， %， ++， --
- 对于/，两个整数相除，结果依然是整数
- 对于%，只有整型变量可以进行取模运算
- 对于++ --，前置是先让变量++或--，然后进行表达式运算，后置则反之

### 2、赋值运算符
- 用于将表达式的值赋给变量
- =， +=， -=， *=， /=， %= 

### 3、比较运算符
- 用于表达式的比较，并返回一个布尔值
- ==， !=， <， >， <=， >=

### 4、逻辑运算符
- 用于根据表达式的值返回布尔值
- !， &&(同真为真，其余为假)， ||(同假位假，其余为真)
